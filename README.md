# 最终幻想 VII 重生 | FINAL FANTASY VII REBIRTH

> 酷炫的赛博朋克风格游戏官网

![FFVII Rebirth](https://images.unsplash.com/photo-1518709268805-4e9042af9f23?w=1920&q=80)

## ✨ 特性

- 🌌 **粒子动画背景** - 80个浮动粒子，自动连接形成网络
- ✨ **鼠标跟随光效** - 霓虹蓝色光晕跟随鼠标
- 🎬 **加载动画** - 进度条 + 脉冲效果
- 🎠 **角色轮播** - 自动播放 + 手动切换 + 键盘导航
- 🏔️ **视差滚动** - 背景图层深度移动
- 📊 **滚动触发动画** - 元素进入视口时平滑出现
- 🎨 **霓虹赛博朋克配色** - 青色 + 紫色 + 粉色
- 📱 **响应式设计** - 完美适配各种屏幕

## 🎮 页面结构

- **Hero 区域** - 霓虹标题 + 浮动光效 + 评分展示
- **故事区域** - 引用风格 + 背景图片
- **角色轮播** - 4个角色卡片，自动/手动切换
- **世界地图** - 4个地点卡片，视差背景
- **战斗系统** - 6个特性卡片，图标动画
- **媒体评分** - 圆形进度 + 徽章展示
- **CTA 区域** - 购买按钮 + 平台信息

## 🛠️ 技术栈

- **HTML5** - 语义化标签
- **CSS3** - Flexbox, Grid, Animations, Variables
- **JavaScript** - Vanilla JS, Canvas API
- **Fonts** - Google Fonts (Cinzel, Orbitron, Noto Sans SC)
- **Images** - Unsplash 高质量图片

## 📝 开发

```bash
# 本地预览
cd ff7rebirth-premium
python3 -m http.server 8888
# 访问 http://localhost:8888
```
